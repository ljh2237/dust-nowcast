数据与模型架构

理论机制层

源区条件 → 起沙条件 → 输送条件 → 落区影响

数据层

站点时序 + 背景场 + 静态地理因子 + 遥感扩展位

模型层

时间编码 + 空间图编码 + 多任务输出

业务层

风速 / 风险等级 / 预警概率 / 可解释性 / 决策支持

项目必要性与限定条件

  • 区域限定：聚焦中国西北重点沙源区与下游走廊
  • 时效限定：0–6 小时短临增强预测
  • 数据限定：多源数据条件下优势更显著
  • 工程限定：轻量、可解释、可部署
  • 决策定位：最后一公里辅助决策增强
  • 不替代国家/省级业务预报
  • 适合答辩展示、项目申报、客户演示
  • 支持官网展示 + 在线体验分流模式
