机制约束下的多源时空融合沙尘暴短临预测系统

中国西北 · 0-6 小时短临预测 · 风速 + 风险等级 + 预警概率 + 可解释性

产品定位

  • 区域：河西走廊—宁夏北部—陕北西段
  • 用户：政府、交通、农业、园区、企业
  • 角色：国家/省级业务预报“最后一公里”增强

访问方式

  • 仓库主页：github.com/ljh2237/dust-nowcast
  • 本地动态产品：python scripts/run_api.py + python scripts/run_streamlit.py
  • 接口文档：http://127.0.0.1:8000/docs

输出能力

  • 未来 1h/3h/6h 风速预测
  • 沙尘风险等级分类
  • 预警阈值触发概率（含置信区间）
  • 变量级/时间级/空间级解释结果
