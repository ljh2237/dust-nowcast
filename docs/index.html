核心能力

多源数据融合

站点观测 + 背景场 + 静态地理因子，支持数据缺失回退与版本记录。

时空联合建模

时间注意力 + 图注意力，建模站点之间传播与空间依赖。

多任务与不确定性

1/3/6h 风速、风险、预警概率，输出 P10-P90 置信区间。

可解释与可部署

变量/时间/空间三层解释，支持 Docker 与多平台部署。

适用场景

  • 政府应急：高风险时段提前预警与联防联控支持
  • 交通运输：路网、枢纽、车队沙尘风险管理
  • 农业生产：设施农业与春播关键时段风险提醒
  • 园区企业：作业排程与安全阈值决策辅助

限定条件与边界

  • 适用于中国西北区域级 0–6h 短临增强预测
  • 要求区域具备可获取站点/背景场/静态因子
  • 当前事件标签为业务代理标签，不是官方逐时人工标注
  • 不替代国家级业务预报系统
