# Product Overview

## Product Name
机制约束下的多源时空融合沙尘暴短临预测系统

## Core Value
- Region-specific (China Northwest)
- 0-6h short-term nowcasting
- Multi-task and interpretable outputs
- Lightweight private/public deployment

## Target Users
- Government emergency teams
- Transportation and logistics
- Agriculture operators
- Industrial parks
- Enterprises with dust-risk exposure

## Capabilities
- Real-time single station forecast
- Multi-station batch forecast
- Station map risk view
- Historical replay
- Model comparison and explainability
- API and deployment-ready packaging
